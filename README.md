  ğŸ“’ NOTE: ğŸ“’

  ### Building locally
  1. Do this
     Clone from GitHub
      ```sh
      git clone <url>
      ```

  2. Then this  
      ```sh
      run build
      ```

  3. And finally this  
     Do stuff
